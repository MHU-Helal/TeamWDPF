Hello Git Collaborators!

This git repo will have two main branches as main & development. Team members are advised to merge their changes to the development branch from their own branch.

Steps to work with Team WDPF as Git Collaborators:

  1. Step 01: From the git repo page, select the development branch and copy the repo link (https://github.com/MHU-Helal/TeamWDPF.git)
  2. Clone the git repo into your project root folder.
  3. Create a new branch with your name including the module/test_module you might want to work on.
  4. Make sure your current working branch is your own branch. (Git checkout branchName to switch between branches.)

    5. [Once you make changes to your branch/module follow the below steps.]

  5. Use [git add .]
  6. Use [git commit -m "Your message"]
  7. Use [git push -u origin (your branch name)]

    8. Final Steps:

  8. Go to this link https://github.com/MHU-Helal/TeamWDPF/compare
  9. Select development on the 'base'
  10. Select your own branch on the 'compare'
  11. Create pull request.
  12. Add title & description about your changes
  13. Create pull request
  14. Merge pull request [If no conflicts arise.]
  15. Confirm merge

    16. Now your changes from your own branch will be merged with development branch.

  16. Repeat the steps from git add . until here if you make more changes after that.
